# Hospital Management System


##  How to Get Started
1. **Clone the repo**: Use the "Clone" button in Visual Studio and paste this URL.
2. **Setup the Database**: 
   - Navigate to the `Database` folder.
   - Open `HospitalSystem.sql` in SQL Server Management Studio (SSMS).
   - Execute the script to create the database and tables.
3. **Open the Project**:
   - Open `Hospital-System.slnx` in Visual Studio 2022/2026.


## 📁 Folder Structure
- **/Database**: Contains the SQL scripts.
- **/HospitalSystemApp**: Contains the C# Windows Forms source code.
